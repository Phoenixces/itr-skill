# ITR Filing - Document Overview

This folder contains documents that can be used for ITR-2/1 filing (AY 2026-27).
All documents in this folder must be **redacted** - no PAN, Aadhaar, name, account numbers, or any PII. Keep only numbers and figures.

> All personal details below are placeholders. 
---

## Documents Required & Why

| Document | Why Needed |
|---|---|
| **Form 16 Part A** | Confirms gross salary paid per quarter and TDS deducted by employer |
| **Form 16 Part B** | Breaks down salary components, perquisites, deductions claimed by employer, and net taxable salary |
| **Form 12BA** | Itemised perquisite valuation (free meals, gifts, other amenities) under section 17(2) |
| **AIS (Annual Information Statement)** | Cross-check all income — salary, dividends, savings interest, capital gains, MF transactions |
| **Form 26AS / TDS Statement** | Verify TDS credits against employer and other deductors |
| **Broker / Demat statement** | Capital gains — sale price, cost, date for each transaction |
| **MF statement** | MF redemptions — units, NAV, cost, date |
| **Savings bank passbook / statement** | Interest earned across all bank accounts |
| **ESPP Details** | Foreign employer share purchase history — contribution dates, amounts, shares allocated |
| **Dividend confirmation (ESPP broker)** | Foreign dividend amount, foreign withholding tax withheld |
| **Foreign company share price on 31-Mar** | Closing value for Schedule FA foreign asset declaration |
| **Foreign company peak price during FY** | Peak value for Schedule FA |
| **Currency Exchange Rate for FA (31-Mar)** | Convert EUR amounts to INR for Schedule FA and FSI |
| **Self-assessment tax challan** | BSR code, serial, date, amount for Schedule IT |
| **ITR-V / Acknowledgement** | Proof of successful filing — download after e-verification |
